# User Management System

<h2>Description</h2>
<p>This project is a PHP OOP-based User Management System built across 3 versions.</p>

<h2>Technologies Used</h2>
<ul>
  <li>PHP (OOP)</li>
  <li>MySQL / PDO</li>
  <li>HTML & CSS</li>
  <li>XAMPP</li>
</ul>

<h2>Versions</h2>
<ul>
  <li><strong>Version 1:</strong> Abstract classes and interfaces</li>
  <li><strong>Version 2:</strong> Traits and namespaces</li>
  <li><strong>Version 3:</strong> PDO database integration + web interface</li>
</ul>
